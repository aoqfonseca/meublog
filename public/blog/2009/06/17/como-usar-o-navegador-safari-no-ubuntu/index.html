Como Usar O Navegador Safari No Ubuntu?

Jun 17th, 2009

Bem pessoal, esse é daqueles post que uso para não esquecer um procedimento. Venho tendo dificuldades no uso do firefox no Ubuntu pois conteúdo em flash não aparecem ou estão com erro. Por isso parti para testar o Safari… a questão que não encontrei distribuição dele para linux. O jeito foi baixar a versão windows e usar o wine para rodá-lo. Assim o procedimento para isso fica: 1 - Baixar e instalar o wine sudo apt-get install msttcorefonts wine 2- Depois de terminar a instalação, devemos configurar o wine e definir o tipo de windows. Aqui, aconselho, a deixar a versão do Xp. No terminal digite: winecfg 3- Depois disso é necessário copiar as fontes para o diretorio de fontes do wine, Sem isso o sistema irá dar erros consecutivos e não funcionará. cp /usr/share/fonts/truetype/msttcorefonts/Arial*.ttf ~/.wine/drive_c/windows/fonts/ cp /usr/share/fonts/truetype/msttcorefonts/Times_New_Roman*.ttf ~/.wine/drive_c/windows/fonts/ 4- Baixar e instalar o Safari wget http://ubuntu-debs.googlecode.com/files/SafariSetup.exe wine SafariSetup.exe 5- Baixar e instalar o flash player wget http://ubuntu-debs.googlecode.com/files/install_flash_player.exe wine install_flash_player.exe Agora é só aproveitar ao máximo o seu novo browser no seu ubuntu
